### Retrieval Augmented Generation (RAG) App

## Setup


```
cd backend 
```
make a virtual environment and activate it

Install requirements.txt

```
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

```

cd frontend

```
npm install
```

```
npm run dev
```

To access the app go to 
```
http://localhost:5173
```
